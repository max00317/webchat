# webchat
[![Build Status](https://www.travis-ci.org/hua1995116/webchat.svg?branch=master)](https://www.travis-ci.org/hua1995116/webchat)

## 启动项目

MongoDB/Node v7+

```
npm install -----安装依赖
npm run dev -----运行
npm run build -----打包
npm run server ----- 打包后运行
```

## 技术栈

 - 前端 vue，vue-router ,vuex
 - 后端 nodejs，express
 - 数据库 mongodb
 - 通讯 websocket
 - 脚手架工具 vue-cli
